# PSEXEC (запуск от админа)

## Что делает

Небольшая бат-тулза для запуска `cmd.exe` на удаленном Windows-хосте через `PsExec`.

Скрипт:
- спрашивает IP удаленного ПК;
- спрашивает логин;
- спрашивает пароль;
- запускает удаленную `cmd.exe` через `PsExec`.

## Как запускать

```bat
PSEXEC (запуск от админа).bat
```

Порядок такой:
1. вводишь IP удаленного ПК;
2. вводишь учетку с правами администратора;
3. вводишь пароль;
4. ждешь запуск удаленной команды.

## Особенности

- в репозиторий добавлена именно твоя исходная версия скрипта, без правок;
- используется `PsExec`;
- запуск задуман под ручную админскую работу, а не под автоматизацию.

## Что делают ключи

- `-i` — интерактивный запуск;
- `-s` — запуск от имени `SYSTEM`;
- `-u` — учетная запись для подключения;
- `-p` — пароль для этой учетной записи.

## Узкие места

- пароль вводится открыто в окне BAT;
- нужен доступ к удаленному хосту и админские права;
- `PsExec` в некоторых средах может ловиться защитой как remote admin tool;
- если на стороне хоста что-то режет интерактивный запуск, поведение может быть не таким, как ожидаешь.
