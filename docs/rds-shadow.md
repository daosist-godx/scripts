# RDS shadow (запуск от админа).bat

## Что делает

Короткая бат-тулза для shadow-подключения к уже открытой RDP-сессии.

По сути:
- спрашивает IP удаленного ПК;
- показывает сессии через `query session`;
- просит ID нужной сессии;
- запускает `mstsc /shadow` с управлением.

## Как запускать

```bat
RDS shadow (запуск от админа).bat
```

Порядок работы:
1. вводишь IP удаленного ПК;
2. смотришь список сессий;
3. вводишь `RDP user ID`;
4. ждешь запуск окна shadow-подключения.

## Особенности

- тулза максимально простая, без лишней магии;
- сама по себе ничего не меняет в политиках и реестре;
- подходит как быстрый хелпер под ручную админскую работу.

## Узкие места

- удаленный ПК должен быть доступен по сети;
- нужная сессия должна уже существовать;
- у тебя должны быть права на shadow-подключение;
- в части сред пользователь должен подтвердить подключение.
