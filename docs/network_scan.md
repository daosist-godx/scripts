# network_scan.ps1

## Что делает

PowerShell-скрипт для сканирования подсети `/24`.

Скрипт:
- пингует адреса `1..254`;
- пытается определить имя хоста;
- пытается получить текущего пользователя через WMI;
- пытается вытащить `DisplayName` из AD;
- сохраняет результат в `TXT` или `CSV`.

## Как запускать

```powershell
powershell -ExecutionPolicy Bypass -File .\network_scan.ps1
```

Дальше при запуске:
1. введи подсеть в виде `192.168.1` или `10.0.0`;
2. выбери формат: `1` для `TXT` или `2` для `CSV`.

## Особенности

- по умолчанию работает по диапазону `1..254`;
- для другой `/24` код менять не нужно, достаточно ввести другой префикс;
- если нужен другой диапазон, надо менять строку `1..254 | ForEach-Object {`;
- CSV сохраняется с разделителем `;`.

## Узкие места

- `Get-WmiObject` и удаленные запросы могут не работать без прав или при блокировке firewall;
- `Get-ADUser` требует доступного AD-модуля и прав на чтение AD;
- обратное DNS-имя может не определиться;
- поиск идет последовательно, поэтому на больших сетях скрипт может работать долго.
