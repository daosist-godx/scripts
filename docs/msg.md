# MSG (запуск от админа).bat

## Что делает

Простая бат-тулза для отправки сообщения в пользовательскую сессию Windows.

Сценарий такой:
- спрашивает IP удаленного ПК;
- показывает активные сессии через `quser`;
- дает выбрать `Session ID` или `*`;
- шлет сообщение через `msg.exe`.

## Как запускать

```bat
MSG (запуск от админа).bat
```

Как пользоваться:
1. вводишь IP удаленного ПК или оставляешь пусто для локального хоста;
2. смотришь список сессий;
3. вводишь `Session ID` или `*`;
4. пишешь текст сообщения.

## Особенности

- если IP не указан, тулза работает по локальной машине;
- удобно для быстрых алертов пользователю без лишней возни;
- использует `msg /v /w /time:120`.

## Узкие места

- без нужных прав на удаленном ПК команда может не сработать;
- `quser` и `msg` завязаны на доступность хоста и служб терминалов;
- если нужная сессия не видна, сообщение в нее не улетит.
