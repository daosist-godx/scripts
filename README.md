# Scripts

Набор небольших админских утилит.

## `network_scan.ps1`

Сканирует подсеть `/24`, пингует хосты и сохраняет список доступных устройств в `TXT` или `CSV`.

Главное:
- показывает `IP`, `Hostname`, `User`, `DisplayName`;
- диапазон по умолчанию: `1..254`;
- для другой `/24` код менять не нужно, просто введи другой префикс при запуске;
- если нужен другой диапазон адресов, измени строку `1..254 | ForEach-Object {`.

Запуск:

```powershell
powershell -ExecutionPolicy Bypass -File .\network_scan.ps1
```

## `MSG (запуск от админа).bat`

Интерактивная обёртка над `msg.exe` для отправки сообщения в выбранную сессию Windows.

Главное:
- спрашивает IP хоста;
- показывает активные сессии через `quser`;
- отправляет сообщение в выбранный `Session ID` или в `*`;
- если IP оставить пустым, работает с локальным ПК;
- запускать лучше от администратора.

Запуск:

```bat
MSG (запуск от админа).bat
```

Как использовать:
- введи IP удалённого ПК или оставь пусто для локального;
- посмотри `Session ID` в списке;
- введи `Session ID` или `*`;
- введи текст сообщения.
