# EVIL-RDS shadow.bat

Короткий BAT-скрипт для RDP shadow-подключения к активной сессии на удаленном Windows-хосте.

## Что делает

- запрашивает имя или IP удаленного ПК;
- показывает список сессий через `query session`;
- просит ID нужной сессии;
- запускает `mstsc` в режиме shadow с управлением.

## Особенности

- работает только в Windows;
- использует `mstsc /shadow`, `/control` и `/noConsentPrompt`;
- сам по себе не меняет системные политики и ключи реестра;
- `noConsentPrompt` сработает только если это уже разрешено администратором;
- использовать только в легитимных админ-сценариях.

## Запуск

```bat
EVIL-RDS shadow.bat
```
